# Board Game Girl (working title)

A cozy puzzle game built in Godot 4 for Steam.

## Setup
1. Install [Godot 4](https://godotengine.org/) (latest stable)
2. Clone this repo
3. Open Godot → Import Project → select this folder
4. Install GodotSteam addon (see addons/ folder)

## Godot version
Godot 4.x — exact version locked after first editor open.

## Export
Export presets in `export_presets.cfg`. Requires Godot export templates.
Platforms: Windows, macOS, Linux.
